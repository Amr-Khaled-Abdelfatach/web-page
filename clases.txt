         <p>The HTML <code id="html">class</code> attribute is used to specify a class for an HTML element.</p>
         <p>Multiple HTML elements can share the same class.</p><hr>

         <h2>Multiple Classes</h2>

         <p>To define multiple classes, separate the class names with a space, e.g. &lt;div 
         class="city main"&gt;. The element will be styled according to all the 
         classes specified.</p>
